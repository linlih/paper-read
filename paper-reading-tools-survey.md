## 简短观察

- `Zotero / Mendeley / Paperpile / ReadCube` 更偏“长期文献库 + 阅读器 + 引用管理”。
- `Explainpaper / Scholarcy / SciSpace` 更偏“AI 精读和理解加速”。
- `Elicit / Consensus / ResearchRabbit / Semantic Scholar` 更偏“找论文、看证据、做综述、追踪动态”。
- `知云文献翻译 / CAJViewer` 更偏中文学术场景，尤其是英文论文翻译和知网格式兼容。
